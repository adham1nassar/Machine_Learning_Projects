# 📁 Assets Folder

This folder contains all the **datasets, images, and supporting files** used in the machine learning projects within this repository.

- **Images & Plots** – Visualizations generated during model evaluation (e.g., actual vs predicted comparisons).  
- **Model Artifacts** – Supporting files used for analysis and model interpretation.  
- **Notebook References** – Files directly linked to the machine learning notebooks for reproducibility.
